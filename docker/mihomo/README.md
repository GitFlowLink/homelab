# Mihomo (Clash Meta)

Прокси для гибкой маршрутизации трафика на базе Mihomo — форк Clash Meta.

## Архитектура

Устройства в сети
       ↓
Роутер OpenWrt (Cudy WR3000S)
       ↓
Mihomo (прозрачный прокси)
       ↓
Маршрутизация по правилам
    ↙         ↘
Прямое       Через
соединение   прокси сервер

## Протоколы
- VLESS+Reality — основной протокол
- Hysteria2 — резервный, UDP-based, высокая скорость

## Функции
- Прозрачный прокси — все устройства в сети работают автоматически
- Раздельная маршрутизация по IP-CIDR и доменам
- Правила для ресурсов А — прямое соединение
- Правила для ресурсов Б — через прокси
- DNS over HTTPS для защиты DNS запросов
- Поддержка IPv4 и IPv6

## Конфигурация
- Запускается как systemd сервис
- Конфиг в `/etc/mihomo/config.yaml`
- Веб-панель управления на порту 9090
- Автозапуск при старте системы через `systemctl enable mihomo`

## Интеграция
- OpenWrt настроен перенаправлять трафик через Mihomo
- IP-CIDR списки обновляются автоматически
- Fallback на прямое соединение при недоступности прокси
